### 1.什么是mongoDB
- 以JSON为数据模型的文档型数据库。

> 文档型数据库：文档类似于JSON。

- 以下是mongoDB的一个文档
```json
{
    "name":"wt",
    "age":"22",
}
```
- 以下是mongoDB的一个集合
```json
{
    "name":"wt",
    "age":"22",
},
{
    "name":"jy",
    "age":"19",
}
```
### 2.什么是mysql
- MySQL 为关系型数据库(Relational Database Management System), 这种所谓的"关系型"可以理解为"表格"的概念, 一个关系型数据库由一个或数个表格组成。
- 以下为一张表

| id  | name | age |
| - | - | - |
|1|wt|22|
|2|jy|19|
|3|whb|17|
